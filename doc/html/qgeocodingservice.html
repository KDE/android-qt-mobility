Detailed Description

The QGeocodingService class handles all communications to and from a geocoding service.

This is the abstract base class for all classes that handle geocoding and reverse geocoding requests/replies to and from a geocoding service.

QGeocodingReply * QGeocodingService::geocode ( const QGeocodingRequest & request )   [pure virtual]

Submits a geocoding request request and returns the corresponding QGeocodingReply.

Note: Due to the asynchronous nature of requests, you should wait for the QGeocodingService::finished(QGeocodingReply*) signal from this object or the QGeocodingReply::finished() signal from the returned QGeocodingReply object before working with the reply.
