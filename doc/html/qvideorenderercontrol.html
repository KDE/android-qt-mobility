Detailed Description

The QVideoRendererControl class provides a media end point that renders to a video surface.

The interface name of QVideoRendererControl is com.nokia.Qt.QVideoRendererControl/1.0 as defined in QVideoRendererControl_iid.

See also QAbstractMediaService::control() and QAbstractMediaService::setActiveEndpoint().
