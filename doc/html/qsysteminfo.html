Detailed Description

The QSystemInfo class provides access to various general information from the system.

QString QSystemInfo::getVersion ( QSystemInfo::Version type, const QString & parameter = QString() )

Returns the version of QSystemInfo::Version type, with optional parameter as a string.

Note: Version will always be returned in "major.minor.build" format.

In case a particular version does not use the "build" part, it is set to 0. If a particular element is not available at all, an error "Not Installed" will be returned by the API.

Returns true if feature feature is supported, otherwise false.
