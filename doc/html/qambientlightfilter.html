Detailed Description

The QAmbientLightFilter class is a convenience wrapper around QSensorFilter.

The only difference is that the filter() method features a pointer to QAmbientLightReading instead of QSensorReading.

Member Function Documentation

Called when reading changes. Returns false to prevent the reading from propagating.
