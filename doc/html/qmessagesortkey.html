Detailed Description

The QMessageSortKey class defines the parameters used for sorting a subset of queried messages from the messaging store.

A QMessageSortKey is composed of a message property to sort and a sort order. The QMessageSortKey class is used in conjunction with the QMessageStore::queryMessages() function to sort message results according to the criteria defined by the sort key.

For example: To create a query for all messages sorted by their timestamp in decending order:

    QMessageSortKey sortKey(QMessageSortKey::timeStamp(Qt::DescendingOrder));
    QMessageIdList results = QMessageStore::instance()->queryMessages(QMessageFilterKey(), sortKey);

See also QMessageStore and QMessageFilterKey.

Member Function Documentation

QMessageSortKey::QMessageSortKey ()

Create a QMessageSortKey with specifying matching parameters.

A default-constructed key (one for which isEmpty() returns true) sorts no messages.

The result of combining an empty key with a non-empty key is the same as the original non-empty key.

The result of combining two empty keys is an empty key.
