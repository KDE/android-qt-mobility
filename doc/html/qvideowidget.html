Detailed Description

The QVideoWidget class provides a widget which displays video produced by a media object.

Member Type Documentation

enum QVideoWidget::AspectRatio

Specfies how video is scaled with respect to its aspect ratio.

ConstantValueDescription
QVideoWidget::AspectRatioAuto0The aspect ratio is repected, and the video is letter-boxed.
QVideoWidget::AspectRatioWidget1The aspect ratio is ignored and the video stretched to fit the widget.
QVideoWidget::AspectRatioCustom2The video is stretched to a specified aspect ratio and letter-boxed.
