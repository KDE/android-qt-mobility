Detailed Description

The QAccelerationReading class represents one reading from the acceleration sensor.

The acceleration sensor returns acceleration values along 3 axes. The scale of the values is milli-Gs. The axes are arranged as follows.

             +z
              |
              |      +y
              |     /
              |----/----
             /| NOKIA  /|
            //|--/--- / |
           // | /   //  /
          //  |/   //  /
         //   '--------------- +x
        //       //  /
       //       //  /
      /---------/  /
     /    O    /  /
    /         /  /
    ----------  /
    |_________!/

A monoblock device sitting at rest, face up on a desk will experience the force of gravity as approximately -1000 on the Z axis.
