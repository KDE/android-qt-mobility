Member Type Documentation

enum QMediaPlayer::State

Defines the current state of the player object.

ConstantValueDescription
QMediaPlayer::LoadingState0The player object is loading necessary data or libraries to play the content.
QMediaPlayer::PlayingState1The player object is currently playing the content.
QMediaPlayer::PausedState2The player object has paused playing the content, position information is, as necessary, retained.
QMediaPlayer::StoppedState3The player object has stopped playing the content, with stop being requested by the stop() function, position information is retained but will be reset on next call to play()
QMediaPlayer::SeekingState4The player object is currently performing a seek operation.
QMediaPlayer::EndOfStreamState5The player object has stopped playing the content as the content has reached a natural end.
