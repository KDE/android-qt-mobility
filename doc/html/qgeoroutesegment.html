Detailed Description

The QGeoRouteSegment class represents a segment of a route.

A QGeoRouteSegment instance has information about the physcial layout of the route segment, the length of the route and the estimated time and navigation instructions required to traverse the route segment.

Subclasses of QGeoRouteSegment will provide more detailed information about the segment. The type() function can be used to determine the subclass that is being used.

See also QGeoRouteSegment::SegmentType.

Member Type Documentation

enum QGeoRouteSegment::SegmentType

This is used to distinguish between different subclasses of QGeoRouteSegment when they are accessed as QGeoRouteSegment objects.

ConstantValueDescription
QGeoRouteSegment::NormalSegment0The route segment information is not specialized.
QGeoRouteSegment::PrivateTransportSegment1The route segment information is specialized for travel by private transport.
QGeoRouteSegment::PublicTransportSegment2The route segment information is specialized for travel by public transport.
QGeoRouteSegment::TruckSegment3The route segment information is specialized for travel by truck.

See also QGeoRouteSegment::type().
