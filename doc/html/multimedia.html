Multimedia

Multimedia provides a set of APIs that allow the developer to play, record and manage a collection of media content.

  • Overview
  • Reference documentation
    • Main classes
    • Classes for service implementers.

Overview

This library will require Qt 4.6.

To build the library, run qmake and make.

Reference documentation

Main classes

QAudioCaptureSourceInterface to query and select an audio input device
QAudioEncoderControlAbstract class for controling audio encoding parameters, this is provided by a QMediaService object, and is used by QMediaRecorder
QAudioEncoderSettingsSet of audio encoder settings
QAudioFormatStores audio parameter information
QAudioSourceControl and information of the audio source
QCameraInterface for system camera devices
QCameraControl
QCameraExposureControlSupplies control for exposure related camera parameters
QCameraFocusControlSupplies control for focusing related camera parameters
QImageProcessingControlAbstract class for controling image processing parameters, like white balance, contrast, saturation, sharpening and denoising
QMediaControlBase interface for media service controls
QMediaObjectThe base Multimedia object
QMediaPlayerAllows the playing of a media source
QMediaPlayerControlAbstract class for controling media playback, this is provided by a QMediaService object, and is used by QMediaPlayer for playback
QMediaPlaylistList of media content to play
QMediaPlaylistControlAbstract class for controling media playlist navigation, this is provided by a QMediaService object, and is used by QMediaPlayer for playback
QMediaPlaylistProviderAccess to a list of media sources
QMediaRecorderUsed for the recording of media content
QMediaRecorderControlAbstract class for control of media recording
QMediaSourceAccess to the resources relating to a media content
QMetaDataControlAccess to the meta-data of a QMediaService's media
QRadioTunerControlAbstract class for controlling the tuning of analog radio devices
QVideoEncoderSettingsSet of video encoder settings
QVideoWidgetWidget which presents video produced by a media object
