This examples shows how to access published context values from within QML.

The example consists of two programs. The first, battery-provier, is a standard Qt GUI application that emulates a battery. It publishes the keys

    /power/battery/charge
    /power/battery/charging

and provides controls for modifying their values.

The second program, battery-consumer, is implemented in QML with some C++ code implement a battery class and make it to QML.
