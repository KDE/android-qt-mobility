Detailed Description

Property Documentation

FillMode : enum

Set this property to define how the video is scaled to fit the target area.

  • Stretch - the video is scaled to fit.
  • PreserveAspectFit - the video is scaled uniformly to fit without cropping
  • PreserveAspectCrop - the video is scaled uniformly to fill, cropping if necessary

bufferProgress : qreal

This property holds how much of the data buffer is currently filled, from 0.0 (empty) to 1.0 (full).


duration : int

This property holds the duration of the video in milliseconds.

If the video doesn't have a fixed duration (a live stream for example) this will be 0.


error : enum

This property holds the error state of the video. It can be one of:

  • NoError - there is no current error.
  • ResourceError - the video cannot be played due to a problem allocating resources.
  • FormatError - the video format is not supported.
  • NetworkError - the video cannot be played due to network issues.
  • AccessDenied - the video cannot be played due to insufficient permissions.
  • ServiceMissing - the video cannot be played because the media service could not be instantiated.

errorString : string

This property holds a string describing the current error condition in more detail.


hasVideo : bool

This property holds whether the source contains video.


muted : bool

This property holds whether the audio output is muted.


paused : bool

This property holds whether the video is paused.


playing : bool

This property holds whether the video is playing.


position : int

This property holds the current playback position in milliseconds.


read-onlyseekable : bool

This property holds whether position of the video can be changed.


source : url

This property holds the source URL of the video.


status : enum

This property holds the status of video loading. It can be one of:

  • NoMedia - no video has been set.
  • Loading - the video is currently being loaded.
  • Loaded - the video has been loaded.
  • Buffering - the video is buffering data.
  • Stalled - playback has been interrupted while the video is buffering data.
  • Buffered - the video has buffered data.
  • EndOfMedia - the video has played to the end.
  • InvalidMedia - the video cannot be played.
  • UnknownStatus - the status of the video is unknown.

volume : qreal

This property holds the volume of the audio output, from 0.0 (silent) to 1.0 (maximum volume).


Signal Documentation

Video::onBuffered ()

This handler is called when the video stream has finished buffering.


Video::onBuffering ()

This handler is called when the video stream starts buffering.


Video::onLoaded ()

This handler is called when the video source has been loaded.


Video::onPaused ()

This handler is called when playback is paused.


Video::onResumed ()

This handler is called when playback is resumed from the paused state.


Video::onStalled ()

This handler is called when playback has stalled while the video stream buffers.


Video::onStarted ()

This handler is called when playback is started.


Video::onStopped ()

This handler is called when playback is stopped.


Method Documentation

Video::pause ()

Pauses playback of the video.


Video::play ()

Starts playback of the video.


Video::stop ()

Stops playback of the video.
