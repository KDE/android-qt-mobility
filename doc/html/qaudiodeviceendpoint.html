Detailed Description

The QAudioDeviceEndpoint class provides an audio device selector media end point.

The QAudioDeviceEndpoint class provides descriptions of the audio devices available on a system and allows one to be selected as the input or output of a media service.

The interface name of QAudioDeviceEndpoint is com.nokia.Qt.QAudioDeviceEndpoint/1.0 as defined in QAudioDeviceEndpoint_iid.

See also QAbstractMediaService::setAudioInput() and QAbstractMediaService::setAudioOutput().

enum QAudioDeviceEndpoint::DeviceDirection

Identifies if an audio device produces or consumes audio data.

ConstantValueDescription
QAudioDeviceEndpoint::InputDevice0x01The device produces audio data.
QAudioDeviceEndpoint::OutputDevice0x02The device consumes audio data.
QAudioDeviceEndpoint::InputOutputDeviceInputDevice | OutputDeviceThe device both produces and consumes audio data.

enum QAudioDeviceEndpoint::FormFactor
flags QAudioDeviceEndpoint::FormFactors

Identifies the form factor of a device.

ConstantValueDescription
QAudioDeviceEndpoint::UnknownFormFactor0x0000The form factor of the device is unknown.
QAudioDeviceEndpoint::RemoteNetworkDevice0x0001The device outputs to a remote network device.
QAudioDeviceEndpoint::Speakers0x0002The device outputs to speakers.
QAudioDeviceEndpoint::LineLevel0x0004The device renders to a line level socket.
QAudioDeviceEndpoint::Headphones0x0008The device outputs to head phones.
QAudioDeviceEndpoint::Microphone0x0010The devices receives input from a microphone.
QAudioDeviceEndpoint::Headset0x0020The device is a headset.
QAudioDeviceEndpoint::Handset0x0040The device is a handset.
QAudioDeviceEndpoint::UnknownDigitalPassthrough0x0080The device outputs to an unknown digital pass-through.
QAudioDeviceEndpoint::SPDIF0x0100The device outputs to an SPDIF socket.
QAudioDeviceEndpoint::HDMI0x0200The device outputs to an HDMI socket.
QAudioDeviceEndpoint::AllFormFactors~0Mask for all form factors.

The FormFactors type is a typedef for QFlags<FormFactor>. It stores an OR combination of FormFactor values.

enum QAudioDeviceEndpoint::Role
flags QAudioDeviceEndpoint::Roles

Identifies the type of use an audio is appropriate for.

ConstantValueDescription
QAudioDeviceEndpoint::ConsoleRole0x01A device appropriate for system notifications, or game sounds.
QAudioDeviceEndpoint::MultimediaRole0x02A device appropriate for music or video audio.
QAudioDeviceEndpoint::CommunicationsRole0x04A device appropriate for voice or video calls.
QAudioDeviceEndpoint::AllRoles~0Mask for all roles.

The Roles type is a typedef for QFlags<Role>. It stores an OR combination of Role values.
