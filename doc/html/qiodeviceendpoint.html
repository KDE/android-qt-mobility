Detailed Description

The QIODeviceEndpoint class provides a media endpoint that reads or writes a QIODevice.

The interface name of QIODeviceEndpoint is com.nokia.Qt.QIODeviceEndpoint/1.0 as defined in QIODeviceEndpoint_iid.

See also QAbstractMediaService::setDataInput() and QAbstractMediaService::setDataOutput().
