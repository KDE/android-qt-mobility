Detailed Description

The QGalleryRemoveRequest class provides a request which removes documents from a gallery.

Property Documentation

documentId : QString

This property holds the ID of an document to remove from a gallery.

This is equivalent to documentIds with a single ID. If there are multiple documentIds this will be null.

Access functions:

void setDocumentId ( const QString & id )

Notifier signal:

void documentIdsChanged ()

documentIds : QStringList

This property holds a list of IDs of documents to remove from a gallery.

If the list only contains one ID this is equivalent to documentId.
