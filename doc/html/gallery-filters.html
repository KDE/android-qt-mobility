Document Gallery Filters

API for identifying documents within a gallery.

QGalleryContainerFilterFilter which matches content belonging to a container document
QGalleryContainerUrlFilterFilter which matches content belonging to a container document identified by a URL
QGalleryDocumentFilterGallery filter which matches documents with specific IDs
QGalleryDocumentUrlFilterGallery filter which matches documents with specific URLs
QGalleryFilterFiltering criteria for gallery requests
QGalleryIntersectionFilterFilter which matches the intersection of two or more meta-data filters
QGalleryMetaDataFilterFilter which matches documents with meta-data fields containing a specific value
QGalleryMetaDataRangeFilterFilter which matches documents with meta-data fields with values within a specified range
QGalleryUnionFilterFilter which matches the union of two or more meta-data filters
