Detailed Description

The QVideoRendererEndpoint class provides a media end point that renders to a video surface.

The interface name of QVideoRendererEndpoint is com.nokia.Qt.QVideoRendererEndpoint/1.0 as defined in QVideoRendererEndpoint_iid.

See also QAbstractMediaService::createEndpoint() and QAbstractMediaService::setVideoOutput().
