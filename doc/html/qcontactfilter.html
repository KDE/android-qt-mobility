Detailed Description

The QContactFilter class is used to select contacts made available through a QContactManager

This class is used as a parameter to various functions offered by QContactManager, to allow selection of contacts which have certain details or properties.

Member Type Documentation

enum QContactFilter::FilterType

Describes the type of the filter

ConstantValueDescription
QContactFilter::Invalid0An invalid filter which matches nothing
QContactFilter::ContactDetail1A filter which matches contacts containing one or more details of a particular definition with a particular value
QContactFilter::ContactDetailRange2A filter which matches contacts containing one or more details of a particular definition whose values are within a particular range
QContactFilter::ChangeLog3A filter which matches contacts whose timestamps have been updated since some particular date and time
QContactFilter::Action4A filter which matches contacts for which a particular action is available, or which contain a detail with a particular value for which a particular action is available
QContactFilter::GroupMembership5A filter which matches contacts which are members of a particular group
QContactFilter::Intersection6A filter which matches all contacts that are matched by all filters it includes
QContactFilter::Union7A filter which matches any contact that is matched by any of the filters it includes
QContactFilter::IdList9A filter which matches any contact whose id is contained in a particular list of contact ids
QContactFilter::Default8A filter which matches everything
