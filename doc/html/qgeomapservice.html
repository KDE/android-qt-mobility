Detailed Description

The QGeoMapService class handles all communication to and from a geographic map service.

This is the abstract base class for all classes that handle map tile requests/replies to and from a service which provides geographic maps.

QGeoMapTileReply * QGeoMapService::getMapTile ( const QGeoMapTileRequest & request )   [pure virtual]

Submits a map tile request request and returns the corresponding QGeoMapTileReply.

Note: Due to the asynchronous nature of requests, you should wait for the QGeoMapService::finished(QGeoMapTileReply*) signal from this object or the QGeoMapTileReply::finished() signal from the returned QGeoMapTileReply object before working with the reply.
