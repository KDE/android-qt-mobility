    Detailed Description

    The QRotationFilter class is a convenience wrapper around QSensorFilter.

    The only difference is that the filter() method features a pointer to QRotationReading instead of QSensorReading.

    Member Function Documentation

    bool QRotationFilter::filter ( QRotationReading * reading ) [pure virtual]

    Called when reading changes. Returns false to prevent the reading from propagating.

    See also QSensorFilter::filter().
