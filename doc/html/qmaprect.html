QMapRect::QMapRect ( const QGeoCoordinate & topLeft, const QGeoCoordinate & bottomRight, const QPen & pen = QPen(), const QBrush & brush = QBrush(), quint16 layerIndex = 0 )

* Constructs a rectangle with corners defined by its topLeft and bottomRight coordinates. The rectangle outline is drawn with the given pen and the filled with the given brush. The rectangle is displayed at the layer specified by layerIndex.
