Detailed Description

The QContactGroup class provides a collection of contacts that share some commonality. A QContactGroup has a name and consists of a list of ids of contacts who are a member of the group.

Member Type Documentation

enum QContactGroup::Error

This enum specifies an error that occurred during the most recent operation:

ConstantValueDescription
QContactGroup::NoError0The most recent operation was successful
QContactGroup::OutOfMemoryError1The most recent operation failed due to running out of memory
QContactGroup::ContactNotInGroupError2The most recent operation failed because the requested contact is not a member of the group
QContactGroup::ContactAlreadyInGroupError3The most recent operation failed because the specified contact is already a member of the group
QContactGroup::PermissionsError4The most recent operation failed because the caller does not have permission to perform the operation
QContactGroup::UnspecifiedError5The most recent operation failed for an undocumented reason
