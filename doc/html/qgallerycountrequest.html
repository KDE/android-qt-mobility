Detailed Description

The QGalleryCountRequest class provides a request for the number of items in a gallery matching some criteria.

Property Documentation

containerId : QString

This property holds the ID of container item a count request should return the contents of.

This is equivalent to setting a filter of type QGalleryContainerFilter.

If the current filter is not of type QGalleryContainerFilter this will be null.

Access functions:

QString containerId () const

containerUrl : QUrl

This property holds the URL of container item a count request should return the contents of.

This is equivalent to setting a filter of type QGalleryContainerUrlFilter.

If the current filter is not of type QGalleryContainerUrlFilter this will be null.

filter : QGalleryFilter

This property holds a filter identifying the items a request should count.

If no filter is set the results of the request will be determined soley by the itemType property.

Access functions:

itemType : QString

This property holds the type of items a request should count.

If this is not set items of all types will be returned. If no filter is set the number items of this type will be returned.

Access functions:

QString itemType () const
void setItemType ( const QString & type )

live : bool

This property holds whether a the results of a request should be updated after a request has finished.

If this is true the request will go into the Idle state when the request has finished rather than returning to Inactive.
