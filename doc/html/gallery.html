Galleries

QUPnPGallery

A gallery which connects to a UPnP media server.

QMediaGallery

Identical to QDocumentGallery in every way, except it only contains media documents.

QUsbGallery

Gallery of documents on USB connected storage.

This is an alternative to including documents on connected storage devices in the regular document gallery.

This probably also be called something else, and not apply to USB only.

Requests

QGalleryCountRequest

This is equivalent to a QGalleryDocumentRequest with the maximumCount property set to 0, which would return the total number of documents but no actual documents.

This request makes sense if the the performance of the QGalleryDocumentRequest will be improved by removing the totalDocumentCount return value. Inversely if the cost of requesting a sub-set of documents and the total count separately is greater than a combined request then removing totalDocumentCount() from QGalleryDocumentRequest wouldn't be desirable.

    class QGalleryCountRequest : public QGalleryAbstractRequest
    {
    public:
        // Constructors, etc.

        QString contentType() const;
        void setContentType(const QString &type);

        QGalleryFilter filter() const;
        void setFilter(const QGalleryFilter &filter);

        // setFilter overloads.

        // Results
        int totalDocumentCount() const;

    Q_SIGNALS:
        void totalDocumentChanged();
    };
QGalleryImportRequest

Imports a document from a remote source into a gallery. Unnecessary for a gallery on the local file-system, but this could enable a remote gallery to download a document directly.

    class QGalleryImportRequest : public QGalleryAbstractRequest
    {
    public:
        // Constructors, etc.

        // The url to import from.
        QUrl sourceUrl() const;
        void setSourceUrl(const QUrl &url);

        // The id of a container document to import the document into.
        void containerId() const;
        void setContainerId(const QString &id);

    // Results
        QGalleryDocumentList *documents() const;

    Q_SIGNALS:
        void documentsChanged();
    };
QGalleryExportRequest

The same as an import request except the gallery would upload a document it hosts.

    class QGalleryExportRequest : public QGalleryAbstractRequest

QGalleryNewDocumentRequest

Creates a new document in a gallery.

This could be used by a remote gallery to provide a URL for an application to upload a new document to.

    class QGalleryNewDocumentRequest : public QGalleryAbstractRequest
    {
    public:
        // Constructors, etc.

        QString fileName() const;
        void setFileName(const QString &fileName);

        QString mimeType() const;
        void setMimeType(const QString &mimeType);

        QString containerId() const;
        void setContainerId(const QString &id);

        // Results
        QGalleryDocumentList *documents() const;

    Q_SIGNALS:
        void documentsChanged();
    };
