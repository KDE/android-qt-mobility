Detailed Description

Use this class to extract Metadata from a Multimedia object.

Property Documentation

metadataAvailable : const bool

This property holds true if metadata can be read from the Multimedia object.

Access functions:

  • bool metadataAvailable () const

readOnly : const bool

This property holds true if metadata can only be read from the Multimedia object, false if elements can be set.

Access functions:

  • bool isReadOnly () const

Member Function Documentation

QMediaMetadata::QMediaMetadata ( QAbstractMediaObject * mediaObject )

Construct a QMediaMetadata to read and or write Metadata from the Mulitmedia object mediaObject.

QMediaMetadata::~QMediaMetadata ()

Destroys the metadata object.

QList<QString> QMediaMetadata::availableMetadata () const

Returns a list of the names of all the metadata elements available.

QVariant QMediaMetadata::metadata ( QString const & name ) const

Returns the metadata for the element named name.

See also setMetadata().

void QMediaMetadata::metadataAvailabilityChanged ( bool metadataAvailable )   [signal]

Signal the availability of metadata has changed, metadataAvailable will be true if the multimedia object has metadata.

void QMediaMetadata::readOnlyChanged ( bool readOnly )   [signal]

Signal a change in the read only status of meta data, readOnly will be true if metadata elements can not be added or adjusted.

void QMediaMetadata::setMetadata ( QString const & name, QVariant const & value )

Change the value of the metadata element named name, to value.

See also metadata().
