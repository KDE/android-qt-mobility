Detailed Description

The QMessageFolderSortKey class defines the parameters used for sorting a subset of queried folders from the messaging store.

A QMessageFolderSortKey is composed of a folder property to sort and a sort order. The QMessageFolderSortKey class is used in conjunction with the QMessageStore::queryFolders() function to sort folder results according to the criteria defined by the sort key.

See also QMessageStore and QMessageFolderFilterKey.

Member Function Documentation

QMessageFolderSortKey::QMessageFolderSortKey ()

Create a QMessageFolderSortKey with specifying matching parameters.

A default-constructed key (one for which isEmpty() returns true) sorts no folders.

The result of combining an empty key with a non-empty key is the same as the original non-empty key.

The result of combining two empty keys is an empty key.
