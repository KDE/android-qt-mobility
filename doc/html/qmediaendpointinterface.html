Detailed Description

The QMediaEnpointInterface class provides a common interface for media service inputs and outputs.

A media end point is an object that can be set as an input or output of a QAbstractMediaService. End points inherit from both QObject, and QMediaEndpointInterface and are not instantiated directly, instead they are constructed using the QAbstractMediaService::createEndpoint() function of the service they intended to be used with.
