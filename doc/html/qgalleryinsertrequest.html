Detailed Description

The QGalleryInsertRequest class provides a request which inserts existing documents into a gallery.

Property Documentation

documentUrl : QUrl

This property holds the URL of an document to insert into a gallery.

This is equivalent to documentUrls with a single value. If multiple documentUrls are set this will be null.

documentUrls : QList<QUrl>

This property holds a list of URLs to insert into a gallery.

If the list only contains one URL this is equivalent to documentUrl.

sortFields : QStringList

This property holds a list of meta-data fields a request should sort its results on.

Prefixing a field name with the '+' character indicates it should be sorted in ascending order, and a '-' character prefix indicates a descending order. If there is no prefix ascending order is assumed.
