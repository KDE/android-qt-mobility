Detailed Description

The QRouteRequest class represents a request for a route between two points.

This is the base class representing a route request from a given source to a given destination.

void QRouteRequest::addStopOver ( const QGeoCoordinate & stopOver )

Adds the stop over stopOver to the requested route.

The server calculates a route that contains these waypoints (in the given order) as part of the route. These waypoints must be between the destination and the source.
