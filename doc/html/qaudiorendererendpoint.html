Detailed Description

The QAudioRendererEndpoint class provides a media end point that renders audio to a QIODevice

The interface name of QAudioRendererEndpoint is com.nokia.Qt.QAudioRendererEndpoint/1.0 as defined in QAudioRendererEndpoint_iid.

See also QAbstractMediaService::setAudioOutput().
