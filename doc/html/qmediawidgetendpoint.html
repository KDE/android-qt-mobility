Detailed Description

The QMediaWidgetEndpoint class provides a QWidget media output.

The interface name of QMediaWidgetEndpoint is com.nokia.Qt.QMediaWidgetEndpoint/1.0 as defined in QMediaWidgetEndpoint_iid.

See also QAbstractMediaService::createEndpoint() and QAbstractMediaService::setVideoOutput().

Member Type Documentation

enum QMediaWidgetEndpoint::AspectRatio

Defines the width:height ratio to be used for the video.

ConstantValueDescription
QMediaWidgetEndpoint::AspectRatioAuto0Let the decoder find the aspect ratio automatically from the media file (this is the default).
QMediaWidgetEndpoint::AspectRatioWidget1Fits the video into the widget making the aspect ratio depend solely on the size of the widget. This way the aspect ratio is freely resizeable by the user.
QMediaWidgetEndpoint::AspectRatioCustom2The aspect ratio is defined by QMediaWidgetEndpoint::customAspectRatio()
