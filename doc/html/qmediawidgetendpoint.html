Detailed Description

The QMediaWidgetEndpoint class provides a QWidget media output.

The interface name of QMediaWidgetEndpoint is com.nokia.Qt.QMediaWidgetEndpoint/1.0 as defined in QMediaWidgetEndpoint_iid.

See also QAbstractMediaService::createEndpoint() and QAbstractMediaService::setVideoOutput().
