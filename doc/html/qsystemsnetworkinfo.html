Detailed Description

The QSystemsNetworkInfo class provides access to network information from the system.

Member Type Documentation

enum QSystemsNetworkInfo::CellNetworkStatus

This enum describes the status of the network connection:

ConstantValueDescription
QSystemsNetworkInfo::UndefinedStatus-1There is no network device, or error.
QSystemsNetworkInfo::NoNetworkAvailable1There is no network available.
QSystemsNetworkInfo::EmergencyOnly2Emergency calls only.
QSystemsNetworkInfo::Searching3Searching for network.
QSystemsNetworkInfo::Busy4Network is busy.
QSystemsNetworkInfo::HomeNetwork5On Home Network.
QSystemsNetworkInfo::Denied6Network access denied.
QSystemsNetworkInfo::Roaming7On Roaming network.

enum QSystemsNetworkInfo::NetworkMode

This enum describes the type of network:

ConstantValueDescription
QSystemsNetworkInfo::UnknownMode0x00000000No network, or error.
QSystemsNetworkInfo::GsmMode0x00000001Global System for Mobile (GSM) network.
QSystemsNetworkInfo::CdmaMode0x00000002Code division multiple access (CDMA) network.
QSystemsNetworkInfo::WcdmaMode0x00000004Wideband Code Division Multiple Access (W-CDMA) network.
QSystemsNetworkInfo::WlanMode0x00000008Wireless Local Area Network (WLAN) network.
QSystemsNetworkInfo::EthMode0x00000010Wired Local Area network.
