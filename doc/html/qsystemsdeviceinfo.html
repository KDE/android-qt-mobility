Detailed Description

The QSystemsDeviceInfo class provides access to device information from the system.

Member Type Documentation

enum QSystemsDeviceInfo::BatteryLevel

This enum describes the level of the main battery.

ConstantValueDescription
QSystemsDeviceInfo::NoBatteryLevel0Battery level undetermined.
QSystemsDeviceInfo::BatteryCritical1Battery level is critical 3% or less.
QSystemsDeviceInfo::BatteryVeryLow2Battery level is very low, 10% or less.
QSystemsDeviceInfo::BatteryLow3Battery level is low 40% or less.
QSystemsDeviceInfo::BatteryNormal4Battery level is above 40%.

enum QSystemsDeviceInfo::InputMethods

This enum describes the input methods for the device or desktop.

ConstantValueDescription
QSystemsDeviceInfo::KeysOnly0Has only Keys or keyboard
QSystemsDeviceInfo::TouchOnly1Has only touchscreen
QSystemsDeviceInfo::KeysAndTouch2Has both Keys and touchscreen.

enum QSystemsDeviceInfo::PowerState

This enum describes the power state:

ConstantValueDescription
QSystemsDeviceInfo::UnknownPower-1Power error.
QSystemsDeviceInfo::BatteryPower1On battery power.
QSystemsDeviceInfo::WallPower2On wall power.

ConstantValueDescription

QSystemsDeviceInfo::OfflineProfile4Offline profile.

QSystemsDeviceInfo::CustomProfile6Custom profile.

enum QSystemsDeviceInfo::SimStatus

This enum describes the status is the sim card or cards.

ConstantValueDescription
QSystemsDeviceInfo::SimNotAvailable0SIM is not available on this device.
