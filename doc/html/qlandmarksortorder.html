Detailed Description

The QLandmarkSortOrder class defines how a list of landmarks should be ordered according to some criteria.

A QLandmarkSortOrder serves as a base class for various sort orders. If none of the pre-existing sort orders is sufficient then custom filter should be created by inheriting from QLandmarkCustomSort.
