Detailed Description

The QLandmark class describes location or point of interest

It consists of a number of set properties such as name and coordinates as well as a set of generic attributes.

Each QLandmark may be associated with zero or more categories. Categories are manipulated by using QLandmarkCategoryId objects via the setCategories(), addCategory() and removeCategory() functions.
