QMapEllipse::QMapEllipse ( const QGeoCoordinate & topLeft, const QGeoCoordinate & bottomRight, const QPen & pen = QPen(), const QBrush & brush = QBrush(), quint16 layerIndex = 0 )

Constructs an ellipse with bounding box corners specified by topLeft and bottomRight. The ellipse outline is drawn with the given pen and filled with the given brush. The ellipse is displayed at the layer specified by layerIndex.
