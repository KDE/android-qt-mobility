QObject * QAbstractMediaService::createEndpoint ( const char * iid )   [virtual]

Creates a new media end point object from the given iid.

An end point object may be an input or output to a media service and can only be used with the service object that created it.

A list of supported end point interfaces can be queried using the supportedEndpointInterfaces() function.

See also supportedEndpointInterfaces(), setAudioOutput(), and setVideoOutput().

This is an overloaded member function, provided for convenience.

Creates a new media end point object.

QList<QByteArray> QAbstractMediaService::supportedEndpointInterfaces ( QMediaEndpointInterface::Direction direction ) const   [virtual]

Returns a list of interfaces for direction media end point objects that can be created by a media service.
