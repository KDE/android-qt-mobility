Detailed Description

The QAbstractMediaService class provides a common base class for media service implementations.

Member Type Documentation

enum QAbstractMediaService::MediaEndpoint

Enumerates the possible end points a media service may have.

ConstantValueDescription
QAbstractMediaService::VideoInput0A video input end point.
QAbstractMediaService::VideoOutput1A video output end point.
QAbstractMediaService::AudioInput2An audio input end point.
QAbstractMediaService::AudioOutput3An audio output end point.
QAbstractMediaService::StreamInput4A data stream input end point
QAbstractMediaService::StreamOutput5A data stream output end point.
