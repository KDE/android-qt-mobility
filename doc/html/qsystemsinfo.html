Detailed Description

The QSystemsInfo class provides access to various information from the system.

QString QSystemsInfo::getVersion ( QSystemsInfo::Version type, const QString & parameter = QString() )

Returns the version of QSystemsInfo::Version type, with optional parameter as a string.

Note: Version will always be returned in "major.minor.build" format.

In case a particular version does not use the "build" part, it is set to 0. If a particular element is not available at all, an error "Not Installed" will be returned by the API.
