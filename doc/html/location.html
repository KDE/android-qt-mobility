Location API

The Location API provides a library for distributing and receiving location data using arbitrary data sources.

QGeoAddressRepresents an address
QGeoAreaMonitorEnables the detection of proximity changes for a specified set of coordinates
QGeoCoordinateDefines a geographical position on the surface of the Earth
QGeoLocationRepresents a geolocation
QGeoPositionInfoContains information gathered on a global position, direction and velocity at a particular point in time
QGeoPositionInfoSourceAbstract base class for the distribution of positional updates
QGeoRouteRepresentation of a route
QGeoRouteReplyRepresents the result of a request for routing information
QGeoRouteRequestRepresents a request for a route between two points
QGeoRoutingServiceAnages requests to and replies from a geographical routing service
QGeoSatelliteInfoContains basic information about a satellite
QGeoSatelliteInfoSourceAbstract base class for the distribution of satellite information updates
QGeocodingServiceHandles all communications to and from a geocoding service
QManeuverRepresents the maneuver component of a QGeoRouteReply
QMapEllipseRepresents an ellipse that has been added to the map. *
QMapMarkerInternal representation of a map marker
QMapObjectThe base of all map objects
QMapPixmapRepresents a pixmap on a map
QMapPolygonRepresents a polygon that has been added to the map. *
QMapRoute
QMapTileReplyRepresents the response from a mapping service
QMapTileServiceHandles all communication to and from a geographic map service
QMapViewDisplays a map. It supports panning, zooming, different map formats, and the adding of map objects (overlays)
QNmeaPositionInfoSourcePositional information using a NMEA data source
