Detailed Description

The QGeoBoundingBox class defines a rectangular geographic area.

The rectangular region is specified by the upper left and lower right coordinates.

If either of these coordinates are invalid then the QGeoBoundingBox is considered to be invalid as well.

The QGeoBoundingBox class is intended for use with the maps and navigation services. These services assume that the maps will be displayed using the Mercator projection, under which the poles of the map can be asymptotically approached but never reached.

A consequence of this is that is not possible to specify a QGeoBoundingBox which is centred one one of the poles.

Note that this would be possible if the rectangular region was defined by three coordiantes to eliminate potential ambiguity, however being able to specify such a region is not useful in this context.
