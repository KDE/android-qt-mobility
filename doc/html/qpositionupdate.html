Detailed Description

The QPositionUpdate class contains information gathered on one's global position, direction and velocity at a particular point in time.

A QPositionUpdate contains, at a minimum, a geographical coordinate and a timestamp. It may also have heading and speed measurements as well as estimates of the accuracy of the provided data.

See also QPositionSource.

Member Type Documentation

enum QPositionUpdate::Property

Defines the properties for a position update.

ConstantValueDescription
QPositionUpdate::Heading0The bearing to true north, in degrees.
QPositionUpdate::GroundSpeed1The ground speed, in metres/sec.
QPositionUpdate::VerticalSpeed2The vertical speed, in metres/sec.
QPositionUpdate::MagneticVariation3The angle between the horizontal component of the magnetic field and true north, in degrees. Also known as magnetic declination.
QPositionUpdate::HorizontalAccuracy4The accuracy of the provided latitude-longitude value, in metres.
QPositionUpdate::VerticalAccuracy5The accuracy of the provided altitude value, in metres.
