Detailed Description

The QGeoRouteRequestOptions class represents various options which may be specified when requesting geographical routes.

The most common uses of the request methods in QGeoMappingService will specific waypoints and, optionally, the level of detail required for directions.

The QGeoRouteRequestOptions class represents the options corresponding to the more advanced (or less common) options which can be passed to the QGeoRoutingService request methods.
