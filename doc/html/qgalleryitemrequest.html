    Detailed Description

    The QGalleryItemRequest class provides a request for an item from a gallery.

    Property Documentation

    item : QGalleryItemList * const

    This property holds the item returned by a request.

    Access functions:

    QGalleryItemList * item () const

    Notifier signal:

    itemId : QVariant

    This property holds the ID of a item a item request should return.

    Access functions:

    QVariant itemId () const
    void setItemId ( const QVariant & id )

    live : bool

    This property holds whether a the results of a request should be updated after a request has finished.

    If this is true the request will go into the Idle state when the request has finished rather than returning to Inactive.

    Access functions:

    bool isLive () const
    void setLive ( bool live )
