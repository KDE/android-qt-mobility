Document Gallery Requests

API for requesting information, and actions from a document gallery.

QGalleryAbstractRequestBase class for gallery requests
QGalleryCopyRequestRequest which copies documents within a gallery
QGalleryDocumentRequestRequest for a set of documents from a document gallery
QGalleryInsertRequestRequest which inserts existing documents into a gallery
QGalleryMoveRequestRequest which moves documents within a gallery
QGalleryRemoveRequestRequest which removes documents from a gallery
