Detailed Description

The QGalleryDocumentRequest class provides a request for a set of documents from a document gallery.

Property Documentation

containerId : QString

This property holds the ID of container document a document request should return the contents of.

This is equivalent to setting a filter of type QGalleryContainerFilter.

If the current filter is not of type QGalleryContainerFilter this will be null.

documentType : QString

This property holds the type of documents a request should return.

If this is not set documents of all types will be returned. If no filter is set all documents of this type will be returned.

live : bool

This property holds whether a the results of a request should be updated after a request has finished.

If this is true the request will go into the Idle state when the request has finished rather than returning to Inactive.

Access functions:

maximumCount : int

This property holds the maximum number of documents that should be returned by a request.

If this is -1 the results will be unbounded.

By default this is -1.

Access functions:

int maximumCount () const

sortFields : QStringList

This property holds a list of meta-data fields a request should sort its results on.

Prefixing a field name with the '+' character indicates it should be sorted in ascending order, and a '-' character prefix indicates a descending order. If there is no prefix ascending order is assumed.
