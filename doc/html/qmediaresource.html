Detailed Description

The QMediaResource class provides a description of a media resource.

Member Type Documentation

enum QMediaResource::ResourceRole

Identifies the role of a resource in relation to a media item.

ConstantValueDescription
QMediaResource::ContentRole0The resource refers to the content of a media item.
QMediaResource::PreviewRole1The resource refers to a preview of a media item.
QMediaResource::PosterRole2The resource refers to a still image to be displayed in place of an unloaded video.
QMediaResource::CoverArtRole3The resource refers to cover art for a media item.
QMediaResource::ThumbnailRole4The resource refers to a thumbnail image for a media item.

int QMediaResource::audioBitRate () const

Returns the bit rate in bits per second of a media resource's audio stream.

This may be zero if the bit rate is unknown, or the resource contains no audio stream.

See also setAudioBitRate().

QString QMediaResource::audioCodec () const

Returns the audio codec of a media resource.

This may be null if the media resource does not contain an audio stream, or the codec is unknown.

See also setAudioCodec().

int QMediaResource::bitsPerPixel () const

Returns the color depth in bits per pixel of a media resource.

This may be zero if the color depth is unknown, or the resource contains no pixel data (i.e. the resource is an audio stream.

See also setBitsPerPixel().
