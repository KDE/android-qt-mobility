Detailed Description

The QLandmarkIdFilter class is used to search for landmarks based on a list of landmark identifiers.

Member Type Documentation

enum QLandmarkIdFilter::MatchingScheme

Defines the matching behavior of the id filter.

ConstantValueDescription
QLandmarkIdFilter::MatchAll0A landmark must be returned for every id assigned to the filter. Failure to match every id results in a QLandmarkManager::DoesNotExistError.
QLandmarkIdFilter::MatchSubset1Every id does not have to be matched to a landmark. Matching only a subset of the assigned ids is acceptable.
