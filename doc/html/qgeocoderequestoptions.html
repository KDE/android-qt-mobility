Detailed Description

The QGeoCodeRequestOptions class represents various options which may be specified when requesting geocoding information.

The most common uses of the request methods in QGeoCodingService will use a single QGeoCoordinate, QGeoAddress or QString as their argument. A QGeoBoundingBox can also be supplied to bias the results to a particular area.

The QGeoCodeRequestOptions class represents the options corresponding to the more advanced (or less common) options which can be passed to the QGeoCodingService request methods.

See also QGeoCodingService.
