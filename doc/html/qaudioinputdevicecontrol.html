Detailed Description

The QAudioInputDeviceControl class provides an audio input device selector media control.

The QAudioInputDeviceControl class provides descriptions of the audio devices available on a system and allows one to be selected as the input of a media service.

The interface name of QAudioInputDeviceControl is com.nokia.Qt.AudioInputDeviceControl as defined in QAudioInputDeviceControl_iid.
