Detailed Description

The QGeoReply class is the base class for all maps and navigation replies.

This is the base class of all classes that represent replies to (reverse) geocoding, route, and map tile requests.

void QGeoReply::error ( ErrorCode code )   [signal]

This signal is emitted when an error occured. This signal will be emitted after its corresponding QGeoEngine::error() signal. Note: When QGeoNetworkManager is used, the error code will be identical to QNetworkReply::NetworkError.

void QGeoReply::finished ()   [signal]

This signal is emitted when the reply is finished. This signal will be emitted after its corresponding QGeoEngine::finished() signal.

bool QGeoReply::isFinished () const   [virtual]

Returns true if the reply finished and returns false if the reply has not finished yet.
