Detailed Description

The QSystemDeviceInfo class provides access to device information from the system.

Member Type Documentation

enum QSystemDeviceInfo::BatteryLevel

This enum describes the level of the main battery.

ConstantValueDescription
QSystemDeviceInfo::NoBatteryLevel0Battery level undetermined.
QSystemDeviceInfo::BatteryCritical1Battery level is critical 3% or less.
QSystemDeviceInfo::BatteryVeryLow2Battery level is very low, 10% or less.
QSystemDeviceInfo::BatteryLow3Battery level is low 40% or less.
QSystemDeviceInfo::BatteryNormal4Battery level is above 40%.

enum QSystemDeviceInfo::InputMethods

This enum describes the input methods for the device or desktop.

ConstantValueDescription
QSystemDeviceInfo::KeysOnly0Has only Keys or keyboard
QSystemDeviceInfo::TouchOnly1Has only touchscreen
QSystemDeviceInfo::KeysAndTouch2Has both Keys and touchscreen.

enum QSystemDeviceInfo::PowerState

This enum describes the power state:

ConstantValueDescription
QSystemDeviceInfo::UnknownPower-1Power error.
QSystemDeviceInfo::BatteryPower1On battery power.
QSystemDeviceInfo::WallPower2On wall power.

enum QSystemDeviceInfo::Profile

This enum describes the current operating profile of the device or computer.

ConstantValueDescription
QSystemDeviceInfo::UnknownProfile-1Profile unknown or error.
QSystemDeviceInfo::SilentProfile0Silent profile.
QSystemDeviceInfo::NormalProfile1Normal profile.
QSystemDeviceInfo::LoudProfile2Loud profile.
QSystemDeviceInfo::VibProfile3Vibrate profile.
QSystemDeviceInfo::OfflineProfile4Offline profile.
QSystemDeviceInfo::PowersaveProfile5Powersave profile.
QSystemDeviceInfo::CustomProfile6Custom profile.

enum QSystemDeviceInfo::SimStatus

This enum describes the status is the sim card or cards.

ConstantValueDescription
QSystemDeviceInfo::SimNotAvailable0SIM is not available on this device.
QSystemDeviceInfo::SingleAvailable1One SIM card is available on this.
QSystemDeviceInfo::DualAvailable2Two SIM cards are available on this device.
QSystemDeviceInfo::Locked3Device has SIM lock enabled.
