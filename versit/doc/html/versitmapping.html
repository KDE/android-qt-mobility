Introduction

Purpose of this document is to give insight view about how each VCard property mapped to QContact detail. For example N type nmae of VCard will be mapped to QContactName detail.

Unsupported VCard fields

Fields other than mentioned in above table is considered as unsupported at this moment. And will be discarded by QVersitContactImporter. But if API user want to implement a custom support then this fields can be obtained by calling

Example code
