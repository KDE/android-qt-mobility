Overview

Versit API provides a set of methods to parse/read (QVersitReader) and encode/write QVersitWriter versit documents such as vCards (http://www.imc.org/pdi/).

It also provides utilities to import (QVersitContactImporter) QContacts from versit documents and export (QVersitContactExporter) QContacts to versit documents.
