Versit

This example shows how to write a simple versit with the Versit API.

Motivation

This application was written to provide an example to developers of how to use the Qt Versit API. .

[*] vCard is a registered trademark of the Internet Mail Consortium

Use Case

There are applications like Sync and Backup used to import vards into phonebook contacts or export contacts into vcard. So the Sync and Backup application developers can write a code as mentioned in this sample to generate contacts from vcard or generate vcard from contacts.

Interface

The application is aimed at desktop platforms as opposed to mobile platforms.So the whole versit code is developed by using QT libraries and it does not have any dependencies to platform specific implementations.

Known Issues

Only VCard 2.1 version is fully supported

Furthermore:

  • Unsupported X-Fields are not mapped to QContactDetail
